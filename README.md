# Placement Tracker API

A simple REST API built using Spring Boot for managing student placement data.

## Features

* View student details
* Add new students
* JSON-based REST APIs
* Built using Spring Boot

## Technologies Used

* Java 17
* Spring Boot
* Maven
* Git
* GitHub
* Thunder Client

## Endpoints

### Get Students

GET /students

### Add Student

POST /students
